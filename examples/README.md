# LMAS 1.6.3 examples

- **File → Open demonstration** or `lma gui --demo` opens the packaged real-data
  2019-04-30 Oklahoma LMA + GOES-16/GOES-17 GLM hybrid flash.
- `synthetic_demo.lmas-project.yaml` remains a tiny deterministic Project-schema
  example for tests and low-overhead demonstrations.
- `truncate_lma_dat.py` creates time-limited `.dat`/`.dat.gz` files while
  preserving retained source rows exactly and updating essential header fields.
- `generic_network_events.csv` is a normalized ground-network overlay example.
- `startup_profile.lmas-profile.yaml` records startup defaults.
- `figure_batch_manifest.json` and `3d_batch_manifest.json` are batch templates.
